    进行移动开发时的HTML及CSS实践
    掌握移动Web开发在页面架构和布局的方法及差异性
    掌握移动Web开发页面调试的方法

    实现与 设计图（点击查看） 一致的移动端Web页面

    本任务只涉及 HTML 及 CSS

    HTML 及 CSS 代码结构清晰、规范
